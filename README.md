# 📘 Math Docs Editor

A Google Docs–style mathematics writing tool designed for learners and teachers to easily type mathematical content using a built-in strand-based math keyboard.

---

## 🚀 Purpose

To make mathematics writing easier, faster, and more interactive by combining a document editor with a structured mathematics symbol keyboard.

---

## ✨ Features

- Google Docs–style writing interface
- Real-time editable document area
- Built-in mathematics symbol keyboard
- Strand-based structure aligned to curriculum:

### 📚 Covered Strands:
- Numbers
- Complex Numbers
- Probability
- Logarithm
- Currency
- Motion
- Calculus
- Measurement
- Geometry

- Easy insertion of mathematical symbols while typing
- Clean and distraction-free writing space
- Offline-friendly web application

---

## 🧠 Built With

- HTML
- CSS
- JavaScript

---

## 🎯 Goal

To support mathematics learning by making expression, notation, and assignment preparation easier and more interactive.

---

## 🌍 Deployment

This project is designed to run on GitHub Pages.
